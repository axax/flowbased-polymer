Basic flowbased-behaviour Demo

When you press the button, it will trigger iron-ajax because of the connection connector-button-request-ajax which is set on the tap event of the button with @-tap="connector-button-request-ajax" (sender) and the generate-request function on the iron-ajax with ƒ-generate-request="connector-button-request-ajax"

The response event from iron-ajax is connected to the local value {{_response}} with
@-response="((_response))".

The value-observer observes the changes for _request and triggers a value-changed event. Which is connected to call-function via @-value-changed="_response-changed" ---> ƒ-func="_response-changed" and this will trigger the method func of call-function.

Schematics

@-tap="connector-button-request-ajax" ---> ƒ-generate-request="connector-button-request-ajax"
@-response="((_response))" ---> this._response
@-value-changed="_response-changed" ---> ƒ-func="_response-changed"
